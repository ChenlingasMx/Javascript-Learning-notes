1.JavaScript 赋值运算符
赋值运算符（=）向 JavaScript 变量赋值。

2.赋值实例

= 赋值运算符向变量赋值。

赋值
var x = 7;

+= 赋值运算符向变量添加值。

赋值
var x = 7;
x += 8;//15

-= 赋值运算符从变量中减去一个值。

赋值
var x = 7;
x -= 8; //-1


*= 赋值运算符相乘变量。

赋值
var x = 7;
x *= 8; 


/= 赋值运算符对变量相除。

赋值
var x = 7;
x /= 8; 


%= 赋值运算符把余数赋值给变量。

赋值
var x = 7;
x %= 8;  //1
